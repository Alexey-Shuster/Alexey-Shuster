Многопользовательский игровой сервер на C++20. Использует библиотеки Boost (Asio, Beast, Log, Program_Options, JSON, Serialization и др.), PostgreSQL (через libpqxx) и модульную архитектуру для обеспечения расширяемости.<br>
Это серверная часть многопользовательской онлайн-игры, где игроки управляют собаками и собирают добычу. Написанный на современном C++20, сервер рассчитан на высокий уровень параллелизма (пул потоков), расширяемость (подключение библиотек) и надежность работы (ведение журнала, сохранение состояния).
